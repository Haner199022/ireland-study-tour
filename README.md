# CUC × Ireland Study Tour

**Live site:** [https://haner199022.github.io/ireland-study-tour/](https://haner199022.github.io/ireland-study-tour/)

A 12-day Beijing study tour for Irish undergraduates from the Communication
University of China — exploring AI × Chinese traditional culture
international communication.

- `index.html` — main project showcase
- `opening-lecture.html` — Dean Jin's D2 opening address (20 slides)
- `opening-lecture.pdf` — printable version of the address

## Hosting

GitHub Pages serves this site directly from the `main` branch.
Any push to `main` rebuilds the live site automatically.

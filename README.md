# 灵境工坊 · Miracle Craft Workshop

**Live site:** [https://haner199022.github.io/ireland-study-tour/](https://haner199022.github.io/ireland-study-tour/)

A 10-day AIGC application training in Beijing for international students, hosted
by the Faculty of International Media (International Student Affairs Center),
Communication University of China. Text, image, music and video generation;
site visits to Future Education Group and the 798 Art District; a cultural-heritage
team project.

中国传媒大学（境外学生事务中心）主办，面向国际学生的 10 天
AIGC 应用实践培训。

- `index.html` — main program site (schedule · features · fees · closing)
- `share.html` — WeChat share page (QR + summary)
- `share-card.png` — 540×1080 WeChat share image
- `qr.png` — plain QR to the live site
- `opening-lecture.html` / `opening-lecture.pdf` — supplementary opening lecture (predates the official plan; kept for reference)

## Hosting

GitHub Pages serves this site directly from the `main` branch.
Any push to `main` rebuilds the live site automatically.
